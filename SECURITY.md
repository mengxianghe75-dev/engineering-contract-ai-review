# Security Policy

## Reporting Security Issues

Please do not open a public issue for suspected security vulnerabilities.

Report privately via:

- WeChat: mengxianghe75

Please include:

- A short description of the issue.
- Steps to reproduce.
- Affected version or commit.
- Potential impact.

## Sensitive Data

Do not commit real contracts, customer documents, credentials, tokens, private keys, database dumps, or production environment files.

The `backend/uploads/` directory is intentionally kept empty in the public demo repository.
